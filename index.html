※ 동기화 주기: iPhone 약 15분~1시간 / 안드로이드 약 몇 시간마다 자동
📱 iPhone 등록 방법
1. 위 구독 URL 복사
2. 아이폰 설정 앱 열기
3. 캘린더 → 계정 → 계정 추가
4. 기타 → 캘린더 구독 추가
5. 복사한 URL 붙여넣기 → 다음 → 저장
6. 완료! 캘린더 앱에서 UC 일정 확인 ✅
🤖 Android (구글 캘린더) 등록 방법
1. 위 구독 URL 복사
2. PC에서 calendar.google.com 접속
3. 왼쪽 '다른 캘린더' 옆 + 클릭
4. 'URL로 추가' 선택
5. 복사한 URL 붙여넣기 → 캘린더 추가
6. 휴대폰 구글 캘린더 앱에 자동 표시됨 ✅
※ 안드로이드는 PC 구글 캘린더에서 먼저 등록해야 해요
⚠️ 동기화는 휴대폰이 자동으로 처리하며, 즉시 반영되지 않을 수 있어요. 새 일정이 안 보이면 캘린더 앱을 잠시 후 다시 확인해주세요.
